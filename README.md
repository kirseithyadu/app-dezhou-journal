# app-dezhou-journal

本仓库用于归档和发布多个独立 HTML 页面。每个页面为一个独立存档单元，便于浏览、分享与长期保存。

## 项目简介

这是一个以 HTML 页面为单位的归档仓库。所有页面均为静态文件，不依赖后端服务，可直接通过浏览器访问。仓库本身不针对特定域名或网站，仅作为页面集合的存储与发布空间。

## 目录说明

```
app-dezhou-journal/
├── index.html          # 仓库入口页（如有）
├── pages/              # 归档页面目录
│   ├── page-001.html
│   ├── page-002.html
│   └── ...
└── assets/             # 静态资源（如图片、样式、脚本）
    ├── css/
    ├── js/
    └── images/
```

- `pages/`：存放所有归档 HTML 页面。
- `assets/`：存放页面引用的静态资源。

## 页面归档说明

- 每个页面为一个独立归档单元，内容完整，无外部依赖。
- 页面命名遵循 `page-xxx.html` 格式，便于索引。
- 归档页面可能包含文本、图像、链接等元素，但均以本地资源或内联形式呈现，确保长期可访问。
